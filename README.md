# 🏠 Housing-Price-Prediction-MLOps

An end-to-end Machine Learning project to predict housing prices using the Boston Housing dataset.

This project goes beyond model training and focuses on building a complete ML pipeline — from data preprocessing to deployment with CI/CD automation.

## 📌 What This Project Includes

- Data preprocessing & feature engineering
- Linear Regression model training
- Model evaluation (R², MAE, RMSE)
- Flask web application for predictions
- Model deployment
- Heroku cloud hosting
